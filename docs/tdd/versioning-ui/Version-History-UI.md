# Version History UI and Policy

## UI
- Timeline view with snapshots.
- Each entry shows timestamp, author, summary.
- Diff view between versions.

## Policies
- Auto-snapshots every N minutes.
- Major edits trigger snapshot (e.g., large paste, style changes).

## Restore
- Restore creates a new head version.
- User can label versions.

## Collaboration
- Each collaborator's edits show in history.
- Conflict resolution records are visible in history.
