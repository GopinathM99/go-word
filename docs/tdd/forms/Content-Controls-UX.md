# Content Controls UX and Validation

## UI
- Highlight controls on focus.
- Show control title/placeholder.

## Control Types
- Text input
- Checkbox
- Dropdown list
- Date picker

## Validation
- Required fields
- Regex validation
- Date ranges

## Data Binding
- Map controls to XML data sources.
- Preserve bindings for round-trip.

## Editing Rules
- Controls can be locked to prevent deletion.
- Editable if user has permission.
