# Style Editor and Inspector

## Goals
- Allow creation and modification of styles.
- Show effective formatting and overrides.

## Style Inspector
- Shows effective style for selection.
- Lists overrides vs base style.
- Button: "Clear formatting" resets overrides.

## Style Editor
- Create new paragraph/character/table styles.
- Set based-on style.
- Apply to selection or set as default.

## Style Conflict Handling
- If two styles share name, keep unique IDs.
- Show conflicts in UI with rename prompt.

## Live Preview
- Preview style in mini-doc sample.

## Apply Behavior
- Applying style clears conflicting direct formatting unless user chooses to keep.
