# Accessibility Test Plan

## Screen Reader Tests
- NVDA (Windows): navigate headings and paragraphs.
- VoiceOver (macOS): read document content and controls.
- JAWS (Windows): verify toolbar and commands.

## Keyboard-Only Tests
- Full document editing without mouse.
- All toolbar and dialog interactions via keyboard.

## High Contrast Tests
- Windows High Contrast mode.
- macOS Increase Contrast.

## Zoom and Scaling
- 200% UI scaling with no layout breakage.
- Text rendering remains crisp.

## ARIA Audits
- Verify correct roles and labels.
- No missing label warnings.
