# Document Outline and Table of Contents

## Outline View
- Based on paragraph styles (Heading 1-9).
- Tree view with collapsible sections.
- Clicking outline item navigates to heading.

## TOC Generation
- Generated from headings and their levels.
- Updates on demand or auto-update setting.

## TOC Field
- Stored as field with instruction text (e.g., TOC \o "1-3" \h).
- Result runs contain TOC entries.

## Navigation
- Next/previous heading shortcuts.
- Promote/demote heading level via shortcuts.

## Round-Trip
- Preserve TOC field instruction and result text.
- Do not auto-refresh unless user triggers update.
