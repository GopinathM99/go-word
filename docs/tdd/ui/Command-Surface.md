## Goals
- Discoverable commands for mainstream users.
- Fast access for power users.

## Ribbon Tabs and Groups

### Home
- Clipboard: cut, copy, paste, format painter
- Font: font family, size, bold, italic, underline, color, highlight
- Paragraph: align, bullets, numbering, indent, spacing
- Styles: style gallery, new style
- Editing: find, replace, select

### Insert
- Pages: cover page, blank page, page break
- Tables: insert table
- Illustrations: pictures, shapes
- Links: hyperlink, bookmark
- Header/Footer: header, footer, page number
- Text: text box
- Symbols: equation, symbol

### Layout
- Page Setup: margins, orientation, size
- Paragraph: indent, spacing
- Arrange: position, wrap, bring forward/back

### References
- Table of Contents
- Footnotes/Endnotes
- Captions and cross-reference

### Review
- Spelling/Grammar
- Comments
- Track Changes

### View
- View modes: print, draft, outline
- Zoom

## Command Palette
- Unified search for commands.
- Alias support (e.g., "bold" maps to toggle bold).
- Recent commands list.

## Shortcut Matrix (Examples)
- Ctrl/Cmd+B: bold
- Ctrl/Cmd+I: italic
- Ctrl/Cmd+U: underline
- Ctrl/Cmd+Z: undo
- Ctrl/Cmd+Shift+Z: redo
- Ctrl/Cmd+F: find
- Ctrl/Cmd+H: replace
