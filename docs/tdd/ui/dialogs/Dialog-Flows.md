# Dialog Flows

## Open Dialog
- File > Open
- Choose file type filter (DOCX, WDJ, RTF, ODT)
- On open: validate file, load, show progress

## Save As Dialog
- File > Save As
- Select format (DOCX, WDJ)
- Warn on format feature loss

## Export Dialog
- Export to PDF / PDF-A
- Options: page range, scaling, embed fonts

## Print Dialog
- Printer selection
- Page range, copies, duplex
- Print preview

## Style Editor Dialog
- Create/modify style
- Preview sample text
- Apply to document
