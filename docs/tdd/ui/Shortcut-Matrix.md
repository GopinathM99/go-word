# Shortcut Matrix (Draft)

## Core Editing
- Cmd/Ctrl+Z: Undo
- Cmd/Ctrl+Shift+Z: Redo
- Cmd/Ctrl+X: Cut
- Cmd/Ctrl+C: Copy
- Cmd/Ctrl+V: Paste
- Cmd/Ctrl+A: Select all

## Formatting
- Cmd/Ctrl+B: Bold
- Cmd/Ctrl+I: Italic
- Cmd/Ctrl+U: Underline
- Cmd/Ctrl+Shift+>: Increase font size
- Cmd/Ctrl+Shift+<: Decrease font size

## Paragraph
- Cmd/Ctrl+L: Align left
- Cmd/Ctrl+E: Align center
- Cmd/Ctrl+R: Align right
- Cmd/Ctrl+J: Justify
- Cmd/Ctrl+Shift+L: Bulleted list

## Insert
- Cmd/Ctrl+K: Insert/Edit Hyperlink
- Cmd/Ctrl+Shift+S: Insert Symbol

## Navigation
- Cmd/Ctrl+F: Find
- Cmd/Ctrl+H: Replace
- Cmd/Ctrl+G: Go to page

## Review
- Cmd/Ctrl+Shift+E: Track changes toggle
- Cmd/Ctrl+Alt+M: New comment
