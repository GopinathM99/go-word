# Wireflow Overview

This outlines the main UI flows and screen interactions.

## 1. New Document
- Launch app -> template picker -> new document canvas.

## 2. Open Document
- File > Open -> select DOCX -> load -> render -> ready.

## 3. Save / Export
- File > Save / Save As (DOCX, WDJ).
- File > Export (PDF, PDF/A).

## 4. Insert Content
- Insert tab -> image/table/shape -> placed in document.

## 5. Review Flow
- Review tab -> comments/track changes panel.

## 6. Collaboration
- Share -> invite -> presence indicators -> real-time edits.
