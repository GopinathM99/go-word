# Error Codes Catalog

## IPC Errors
- INVALID_COMMAND: Unknown or malformed command.
- INVALID_DOC_ID: Document not found.
- INVALID_PARAMS: Missing or invalid parameters.
- INTERNAL_ERROR: Unhandled exception in core.

## Import/Export Errors
- DOCX_PARSE_ERROR: Malformed OOXML.
- DOCX_UNSUPPORTED: Unsupported feature encountered.
- PDF_EXPORT_ERROR: PDF export failed.
- FILE_WRITE_ERROR: Could not write output file.

## Validation Errors
- SCHEMA_VERSION_MISMATCH
- INTEGRITY_CHECK_FAILED
- MISSING_RESOURCE
